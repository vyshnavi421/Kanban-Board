# Kanban Board
A Trello-style Kanban Board built using HTML, CSS, and JavaScript.
## Features

- Create Task Cards
-  Drag & Drop Cards
-  Persistent Storage using Local Storage
-  Edit Tasks
-  Delete Tasks
-  Task Priority
-  Responsive Design

##  Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage
## How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
